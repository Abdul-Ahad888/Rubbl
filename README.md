# Rubbl - Frontend Showcase Project


## Overview
Rubbl is a responsive, multipage website built using HTML, CSS, JavaScript, and Bootstrap. This project was created as part of my freelance work on Fiverr to demonstrate my front-end development skills. The website features:

- A clean, modern, and fully responsive design.
- Multipage navigation including homepage, product listings, and detailed item pages.
- Interactive UI components and smooth scrolling effects.
- A project layout that adapts seamlessly to desktop, tablet, and mobile devices.
